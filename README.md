**Описание**
* "Место" – это учебный проект в Яндекс.Практикуме. Пока что он состоит из профиля пользователя, который можно редактировать через попап, а так же сетки с фотографиями и их названиями. Можно создавать новые карточки мест, а также удалять их и ставить лайки. Фотографии можно увеличить.
